# SwiftUITemplate
Template repository for SwiftUI projects.

## Environment

|Name|Version|
|:--:|:--:|
|Xcode|13.2.1 (13C100)|
|Swift|5.5.2|
|Deployment Target|iOS 14.0|

## Setup

```
$ make bootstrap
```

## Generate project file

```
$ make project
```

## Generate swift flux files from template.

```
$ make template-flux-<the Module name>
```

e.g.

```
$ make template-flux-UserList
```
